# 🌾 AI Farmer Assistant

An AI-powered web application built with the **MERN Stack** that helps farmers make informed decisions by providing intelligent assistance for crop management, disease detection, weather guidance, and farming-related queries using Artificial Intelligence.

---

## 🚀 Features

* 🤖 AI-powered farming assistant
* 🌱 Crop recommendation based on user inputs
* 🦠 Plant disease detection and suggestions
* 🌦️ Weather insights for better farming decisions
* 💬 AI chatbot for agriculture-related queries
* 📊 Farmer-friendly dashboard
* 🔐 Secure user authentication (JWT)
* 📱 Fully responsive design
* ⚡ Fast and intuitive user interface

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Axios

### Backend Part

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer (Image Upload)

### AI & APIs

* Gemini API 
* Weather API

---

**▶️ Run the Project**

-> Start Backend
* cd backend
* npm run dev
-> Start Frontend
* cd frontend/ai-farmer-assistant
* npm run dev
